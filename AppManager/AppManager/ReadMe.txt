
AppManager is the tool to help you launch your programs easily.
This program is open source, free software, distributed by Microsoft Public License (Ms-PL).
If you do not accept the license, do not use the software.

AppManager's codeplex page: http://appmanager.codeplex.com.
AppManager will download official updates from http://dimanick.ru/MySoft/AppManager/Update.

To run this program you will need to install .Net framework version 3.0.
It is recommended to install .Net framework version 3.5 from here 
http://www.microsoft.com/downloads/details.aspx?familyid=AB99342F-5D1A-413D-8319-81DA479AB0D7&displaylang=en.
